## 用法

拖入图片，鼠标按下与松开切换最后拖入的两张图片。默认自动按高度对齐。

拖放图片至 exe 可以直接打开比较。

快捷键 `Esc` 或者 `Ctrl+W` 退出。

鼠标点击图片或者按 `空格键` 快速查看另一张。

快捷键 `S` 交换查看两张图片。
